Welcome to my Yelp Sentiment Analysis Project

Created by Chinmay Mangalwedhe

This is a project that I made that scrapes Yelp pages and gives a sentiment analysis using the pretrained neural network BERT. After it gives a sentiment to every review on the Yelp page provided by you, it will create a histogram plot and summary statistics of the sentiments collected.

Visit this link if you would like to learn more about what BERT is and what it can offer: Article by Rani Horev

As this scrapes Yelp's webpage (to capture all reviews) and gives a sentiment to each rating, this may take a while so please do not leave the webpage.

In the future, I plan to expand this and include Google Reviews.
